feat: v5.1 - Theme toggle with system preference + Original language option

Features:
- Dark/Light theme toggle (follows system preference by default)
- Theme persists in localStorage
- Listens for system theme changes
- Language options: Original (default), EN, CN
- "Original" shows content as-is without translation

Technical:
- CSS variables for dark/light themes
- [data-theme="light"] selector for light theme
- window.matchMedia listener for system theme
- Language toggle with 3 options
